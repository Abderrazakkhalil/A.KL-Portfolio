# Khalil Abderrazak Portfolio

Personal portfolio based on a modern interactive React template, customized for Khalil Abderrazak.

## Stack
- React + TypeScript + Vite
- Tailwind CSS
- Framer Motion
- React Router

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Main Content
- Home: positioning and navigation
- Projects: RoboNavX, AI CV Generator, Smart POS
- Experience & Skills: internships, timeline, technical stack
- Leadership & Contact: clubs, creative activities, achievements, contact
